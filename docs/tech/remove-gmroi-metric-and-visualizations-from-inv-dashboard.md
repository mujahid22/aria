# Remove GMROI metric and visualizations from INV dashboard
## Overview
This Business Requirements Document (BRD) outlines the need to remove the GMROI (Gross Margin Return on Investment) metric and all its associated visualizations from the Inventory (INV) dashboard. The goal is to streamline the dashboard view by eliminating redundant or less critical metrics without affecting other functionalities.
## Description
The INV dashboard currently includes the GMROI metric and its associated visualizations. To improve clarity and focus, this metric and all related visualizations must be removed. This change ensures that users are not distracted by less critical data and that the dashboard remains performant and easy to navigate.
## Acceptance Criteria
- GMROI metric is no longer visible on the INV dashboard.
- All visualizations tied to the GMROI metric are removed from the INV dashboard.
- No other metrics or visualizations on the INV dashboard are affected.
- The removal of GMROI does not break any dependencies or cause errors in the dashboard.
- Users can no longer filter or interact with GMROI-related data in the INV dashboard.
- The change is deployed to production and verified by QA.
## Priority
Medium
