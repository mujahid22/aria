# Add Dealer Name Field to Inventory Report
## Overview
This Business Requirements Document (BRD) outlines the need to extend the Inventory report functionality to include a Dealer name field. This enhancement will improve traceability and clarity by associating inventory items with their respective dealers.
## Description
The current Inventory report system does not include a field for capturing the Dealer name associated with inventory items. This feature request aims to extend the Inventory report input form to include a Dealer name field, ensure the data is saved and displayed in the report output, and support editing and exporting the Dealer name.
## Acceptance Criteria
- The Inventory report input form includes a field to enter the Dealer name.
- The Dealer name is saved and associated with the inventory item.
- The Dealer name is displayed in the Inventory report output.
- The Dealer name field supports alphanumeric characters and common punctuation.
- The Dealer name is included in the report export (e.g., PDF/CSV) if applicable.
- Editing an inventory item allows updating the Dealer name.
## Priority
High
## Implementation Notes
- How will the Dealer name field be validated to ensure it only accepts alphanumeric characters and common punctuation?
- How will the Dealer name field be displayed in the Inventory report output?
- How will the Dealer name field be included in the report export (e.g., PDF/CSV)?
