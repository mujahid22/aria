# Add Sub-Category Filter to Inventory Dashboard
## Overview
This Business Requirements Document (BRD) outlines the need to enhance the Inventory Dashboard by adding a sub-category filter. This feature will enable users to perform more granular data analysis and reporting by filtering inventory items based on sub-categories.
## Description
The Inventory Dashboard currently allows users to filter items by broad categories. To improve usability and provide deeper insights, a sub-category filter must be introduced. This filter will allow users to refine their view of inventory data by selecting one or multiple sub-categories, enabling more precise analysis and reporting.
## Acceptance Criteria
- The Inventory Dashboard must include a new filter dropdown or selector for sub-categories.
- The sub-category filter must display all available sub-categories relevant to the selected or default category.
- Users can select one or multiple sub-categories to filter the dashboard data.
- The dashboard data must update in real-time or near-real-time to reflect the selected sub-category filters.
- The sub-category filter must be persistent across dashboard refreshes or user sessions until explicitly changed.
- The filter must not break existing functionality or performance of the Inventory Dashboard.
