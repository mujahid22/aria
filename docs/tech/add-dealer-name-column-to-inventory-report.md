# Add Dealer Name Column to Inventory Report
## Overview
This Business Requirements Document (BRD) outlines the need to enhance the Inventory report by adding a 'Dealer Name' column. This improvement aims to provide better traceability and usability by displaying the dealer associated with each inventory item.
## Description
The current Inventory report does not include the Dealer name, which limits visibility and traceability of inventory items. This requirement involves adding a 'Dealer Name' column to the report to ensure users can easily identify the dealer associated with each item. The column will be integrated seamlessly into all views of the report, including web, export, and print formats.
## Acceptance Criteria
- The Inventory report must include a new column labeled 'Dealer Name'.
- The 'Dealer Name' column must display the correct name associated with each inventory item.
- The column should be visible in all views of the Inventory report (e.g., web, export, print).
- The addition of this column must not break existing functionality or layout of the report.
- The 'Dealer Name' data must be sourced from the existing dealer database or system of record.
- The column should be sortable and filterable if the report supports these features.
## Priority
High
