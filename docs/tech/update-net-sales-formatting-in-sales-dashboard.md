# Update Net Sales Formatting in Sales Dashboard
## Overview
This Business Requirements Document (BRD) outlines the need to enhance the net sales formatting in the Sales Dashboard. The goal is to improve clarity, consistency, and visual presentation to support better user understanding and decision-making.
## Description
The current net sales formatting in the Sales Dashboard lacks consistency and visual clarity, which can hinder user interpretation and decision-making. This update aims to standardize the formatting of net sales values, ensuring uniformity in number representation, currency symbols, decimal places, and visual elements like fonts, colors, and alignment. The changes will enhance the overall user experience and ensure accurate data representation.
## Acceptance Criteria
- Net sales values must use a consistent number format (e.g., thousands or millions) with clear labeling.
- Currency symbols and decimal places must be uniformly applied across all net sales displays.
- Visual elements like fonts, colors, and alignment must be standardized for net sales data.
- The updated formatting must not distort or misrepresent the underlying data.
- Users must be able to easily distinguish net sales from other metrics (e.g., gross sales, discounts).
- The changes must be tested and confirmed to work across all supported devices and screen sizes.
