# Remove GMROI Metric from Inventory Dashboard
## Overview
This Business Requirements Document (BRD) outlines the need to remove the GMROI (Gross Margin Return on Investment) metric from the Inventory (INV) Dashboard. The goal is to simplify the dashboard view by eliminating unnecessary clutter while ensuring the integrity and functionality of remaining metrics and data points.
## Description
The GMROI (Gross Margin Return on Investment) metric is currently displayed in the Inventory (INV) Dashboard. This metric is deemed non-essential for the primary users of this dashboard, and its removal will streamline the view, improve usability, and reduce cognitive load for users. This change does not impact any other metrics or data points in the dashboard.
## Acceptance Criteria
- GMROI metric is no longer visible in the Inventory (INV) Dashboard.
- No other metrics or data points in the dashboard are affected by this removal.
- The dashboard layout remains intact and functional after the removal.
- Users can no longer select or interact with the GMROI metric in the dashboard.
- The change is verified in the production environment post-deployment.
